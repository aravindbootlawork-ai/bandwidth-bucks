
# BandwidthBucks

Turn your idle internet into real cash. Monetize your unused bandwidth securely and easily.

## Core Features
- **Real-time Monetization**: Share idle bandwidth and earn INR via a live simulation.
- **AI Insights**: Personalized earnings strategy generated by GenAI.
- **Secure Payouts**: Withdraw via UPI or PayPal (minimum thresholds apply).
- **Admin Panel**: Manage users and process payout requests.
- **Cross-Platform**: Ready for background node integration (APK/EXE placeholders).

## Deployment Instructions

### Option 1: Firebase (Recommended for integrated Auth/Firestore)
To deploy to Firebase Hosting (Note: Next.js SSR requires the Blaze plan for Cloud Functions):
1. **Install Firebase CLI**: `npm install -g firebase-tools`
2. **Login**: `firebase login`
3. **Init**: `firebase init` (Select Hosting and Firestore)
4. **Deploy**: `firebase deploy`

### Option 2: Netlify
The project includes a `netlify.toml` for easy deployment:
1. Push your code to a Git repository (GitHub/GitLab/Bitbucket).
2. Connect your repository to Netlify.
3. Add your Firebase environment variables (API Key, Project ID, etc.) to the Netlify Site Settings.
4. Netlify will automatically detect the Next.js project and deploy it.
